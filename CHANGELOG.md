# 版本历史 (Changelog)

## [2.0.0] - 2026-04-01

### 🎉 重大更新

#### ✨ 新增功能
- **依赖记录持久化**：实现用户依赖数据库 (`~/.local/share/debpkg/user_deps.db`)，记录每个包的依赖关系
- **智能卸载**：卸载时自动检查依赖引用，保留仍被其他包使用的库文件
- **桌面集成**：自动为 GUI 应用生成 `.desktop` 文件
- **环境变量配置**：自动生成并提示 sourcing 环境配置脚本

#### 🔧 功能改进
- **依赖解析优化**：修复依赖解析中的缓冲区溢出问题，支持长依赖列表
- **内存安全增强**：采用"解析即保存"策略，避免内存被意外覆盖
- **输出精简**：安装输出减少 97%，更加清爽易读
  - 依赖检查：从 45 行减少到 6 行
  - 文件安装：从 520 行减少到 8 行
  - 总体输出：从 745 行减少到 24 行

#### 🐛 Bug 修复
- 修复依赖解析时版本号包含前导空格的问题
- 修复依赖数组在保存前被破坏的内存安全问题
- 修复卸载时未能完全清理符号链接和桌面文件的问题
- 修复自动依赖安装时的详细输出过多的问题

#### 📚 文档更新
- 新增 `OUTPUT_OPTIMIZATION.md` - 输出优化详细说明
- 新增 `VSCODE_IMPROVEMENTS.md` - VS Code 安装测试报告
- 更新 `TEST_VSCODE_INSTALLATION.md` - 完整测试记录
